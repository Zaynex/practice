先添加一个背景图，让用户感觉在加载，等到imgload完全的时候再添加图片,并且设置了透明度的动画，感觉这个流程特别和谐

Mars

    $(function() {
    $('.post img').each(function() {
       var el = this;
       var image = new Image();
       image.src = el.src;

       image.onload = function() {
         $(el).parent().addClass('loaded');
       }
    })
  })
