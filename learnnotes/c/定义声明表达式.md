定义和声明之间的关系：
如果一个声明同时也要求分配存储空间，则称为定义。

在 JS 中声明一个变量就是：
```
let a
```
定义一个变量则为：
```
let a = 10
```

不过函数声明也同时是定义。
```
function foo(){
    //
}
```

### 变量初始化
所有的变量名都代表一块存储空间。
定义一个变量，就是分配一块存储空间并给它命名。给一个变量赋值，就是把一个值存到了这块存储空间中。

### 表达式
操作数加上运算符组成表达式


```
printf("%c", 'a' + 1)
// b
```
'a' 在ASCII码中对应的数字是 97，再 + 1 得到 98。因为是通过 `%c` 去打印，所以需要转换成字符型，打印成 `b`
