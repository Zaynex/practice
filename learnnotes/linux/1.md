### cd 切换目录
- 绝对路径： 以 `/` 开头的是从根目录开始切
- 相对路径： 以 `./`开头的是从当前目录开始

*注意，termainl 已经默认省略了`./`*

- cd 切换到home目录x
- cd -: 切换到先前工作目录（应该就是上一级目录吧）
- cd ~username: 切换目录到某个用户的home目录


### ls
- ls -a: 列出隐藏的 `.文件`， 如 `.gitignore`
- ls -lt: 列出文件，包括时间格式，以及读写权限等

```
-rw-r--r--
```
开头的`-` 表示一个普通文件，如果是`d` 说明是一个目录
后面三个字符串表示的是文件的读写权限
